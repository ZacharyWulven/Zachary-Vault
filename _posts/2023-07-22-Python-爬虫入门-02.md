## 17 xpath

### 安装步骤

1. 安装 xpath
* chrome 安装 xpath helper 插件
* xpath 快捷键 `Ctrl + Shift + X`
2. 安装 lxml

```bash
$pip3 install lxml
```

3. 导入 etree

```python
from lxml import etree
```

###  xpath 解析
1. 解析本地文件：使用 `etree.parse`
2. 解析服务器响应文件：`etree.HTML(response.read().decode('utf-8'))`

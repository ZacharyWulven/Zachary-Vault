# 四 Java 对象与类

## 类

> Java 编写的所有代码都在某个类中，`Object` 是超类，所有类都扩展自 `Object`
{: .prompt-info }

## 对象
* 状态（state）：每个对象都会保存着描述当前状况的信息。
    - 对象状态改变必然是调用方法的结果，如不经调用就改变状态，说明破坏了封装性
* 标识（identity）：如何区分有相同行为和状态的不同对象
* 对象行为（behavior）：可以对这个对象做哪些操作，或可以对这个对象应用哪些方法？
